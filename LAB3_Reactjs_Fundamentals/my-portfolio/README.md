# Portfolio Website

## Description
Portfolio website that shows past projects.

## Technologies Used
- React 18
- Vite
- CSS3
- Lucide React Icons
- npm install @fortawesome/fontawesome-free

## Features
- Responsive design
- Dark/Light theme switcher
- Project showcase with filtering
- Contact form with validation
- Smooth scrolling navigation
- icon
